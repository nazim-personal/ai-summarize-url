# AI-Driven Website Summariser (React + TypeScript)

## Description
A frontend-only React web application built with TypeScript that summarizes
public webpages using an AI model.

## Features
- URL input
- Webpage content fetching
- AI-generated summary
- Loading indicator

## Tech Stack
- React
- TypeScript
- GROQ API

## Setup
npm install
npm run dev

## AI Usage
Webpage content is extracted in the browser and sent to an AI model to
generate a concise summary.
